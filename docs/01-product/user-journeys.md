# Ścieżki Użytkownika (User Journeys)

**Status**: Active
**Ostatni przegląd**: 2026-02-18
**Właściciel**: Deweloper / AI

---

System StageBrain obsługuje trzy fazy pracy operatora/showcallera z koncertem:

## Fazy

### 1. Pre-show (Konfiguracja)
Operator przygotowuje system przed koncertem: wybiera/importuje setlistę, konfiguruje venue i kalibrację, ustawia curfew, testuje audio.

### 2. Live (Koncert)
Operator korzysta z panelu w czasie rzeczywistym: monitoruje engagement, zarządza segmentami, przegląda rekomendacje ML, kontroluje czas, dodaje tagi.

### 3. Post-show (Analiza)
Operator przegląda dane po koncercie: timeline engagement, statystyki segmentów, log decyzji, eksport danych, generowanie raportów.
