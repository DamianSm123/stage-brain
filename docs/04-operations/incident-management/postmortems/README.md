# Postmortems Archive

Folder ten służy do archiwizacji raportów z incydentów (Postmortems).

## Konwencja nazewnictwa plików
`YYYY-MM-DD-krotki-opis-incydentu.md`

Np.:
*   `2026-02-15-queue-backlog-outage.md`
*   `2026-03-01-database-connection-limit.md`

## Proces
1.  Po zakończeniu incydentu, osoba odpowiedzialna kopiuje szablon [postmortem.md](../../../06-templates/postmortem.md) do nowego pliku.
2.  Wypełnia raport wspólnie z zespołem (np. na spotkaniu post-incident review).
3.  Zatwierdzony raport trafia do tego folderu w repozytorium.
